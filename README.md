# CreatorPulse AI Assistant
A production-ready, hands-free conversational AI companion built to streamline creative workflows and eliminate writer's block for digital content creators.

## 💡 The Real-World Inspiration
As a creator managing a YouTube channel, moving between video editing setups and clunky, browser-based AI writing interfaces kills creative flow. CreatorPulse solves this bottleneck by putting an advanced content strategy brain directly into Telegram—enabling on-the-go scripting, title ideation, and rapid hook generation right from a pocket.
## 🛠️ Tech Stack & Architecture
* *Frontend UI:* Telegram Bot API for low-latency, mobile-first user interaction.
* *Orchestration Core:* n8n workflow automation platform handling live trigger events.
* *Brainpower:* Google Gemini Advanced Agent framework optimized for plain-text response generation.
* *State Management:* Native n8n chat memory nodes to preserve context across continuous conversations.
## 🚀 Key Automation Features
1. *Automated Execution:* Runs 24/7 independently of local machines via background listeners.
2. *Context-Aware Memory:* Remembers preceding prompts, allowing interactive script revisions (e.g., "Write out a script layout for hook option #2").
3. *Optimized Output Constraints:* Deep system prompting forces clean plain-text payloads, completely bypassing parsing bugs and text cutoff issues.
